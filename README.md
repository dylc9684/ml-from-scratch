# Algorithm Zoo

An interactive React-based AI playground for browsing algorithms, tuning parameters, running in-browser simulations, uploading datasets, and studying synchronized Python and JavaScript implementations.

## Architecture

- React + TypeScript + Vite for the frontend shell.
- JavaScript/TensorFlow.js execution in the browser for live runs and animation state.
- Canvas + D3 scales for high-performance algorithm visualizations.
- KaTeX for mathematical notation.
- Prism.js for the dedicated dual-language `CodeViewer`.
- Python code is educational and synchronized with UI parameters; JavaScript remains the runtime execution path.

## Commands

```bash
npm install
npm run dev
npm run build
```
